System Information

Visual Inventory Tracking System

Platform: NVIDIA Jetson Nano

Core Technology: Computer Vision (OpenCV) & AI

System Cycle

  1. Initialization: System powers on and loads executable files from SD Card.
  2. Input Acquisition: Camera stream initiates (cap->read).
  3. Detection: OpenCV processes frames to locate visual markers.
  4. Verification: System displays processed frames for visual confirmation.
  5. Logging: Discrepancies are logged to SD Card for CDR/FDR reporting.
